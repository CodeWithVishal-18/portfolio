# 🌐 Developer Portfolio
A modern **developer portfolio website** built with **React.js** to showcase my projects, technical skills, and resume.
The portfolio focuses on **clean UI, responsive design, and smooth animations** to provide a professional developer presence.

# 🚀 Features

- Modern responsive UI
- Animated hero section
- About section with technical skills
- Project showcase cards
- Resume preview and download
- Smooth hover animations
- Optimized performance

# ⚙️ Development Process

The portfolio was developed using **React functional components** with a modular architecture.

- Home

4️⃣ Optimized performance using:

# 🚀 Deployment

The project was deployed using **Vercel**.

### Deployment Steps

1. Push project to GitHub

2. Connect repository with Vercel

3. Configure build settings <br>
    Build Command: npm run build
    Output Directory: build
5. Deploy automatically from GitHub.


# ⚠️ Challenges Faced

### 1️⃣ Inconsistent Project Card Layout

**Problem**

Different project images caused uneven card sizes.

**Solution**

Used CSS: <br>
  object-fit: cover;

This ensured images fit perfectly inside fixed containers.

### 2️⃣ PDF Resume Viewer Toolbar

**Problem**

Embedded PDF showed browser toolbar.

**Solution**

Implemented a **custom resume preview card with modal viewer**.

### 3️⃣ Responsive Layout Issues

**Problem**

Some sections broke on smaller screens.

**Solution**

Used:

- Bootstrap grid
- flexible containers
- media queries
